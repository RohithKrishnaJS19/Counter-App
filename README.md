# Counter-App
This repository contains my Counter App using React.
A simple and interactive Counter Application built using React.js.
This project demonstrates the use of React Hooks (useState), event handling, and basic UI styling.

# Features

•Increase the counter
•Decrease the counter
•Reset the counter
•Shows message when minimum limit is reached
•Responsive UI with background image

# Tech Stack

* React.js
* JavaScript (ES6)
* CSS3
* HTML5
